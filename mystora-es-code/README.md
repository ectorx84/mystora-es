# Mystora ES — Guía personalizada

Versión española de Mystora. Perfil astrológico y numerológico personalizado.

## Stack
- Next.js (App Router) / React / Tailwind CSS
- Claude API (Haiku teaser / Sonnet informe)
- Stripe Checkout
- Brevo (email)
- Vercel Blob (almacenamiento informes)
- Vercel (hosting)

## Deploy
Auto-deploy via Vercel desde branch `main`.
